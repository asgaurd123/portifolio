## Run Locally

Clone the project

```bash
  git clone https://github.com/asgaurd123/portifolio.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  pip install django
```

Start the server

```bash
  python manage.py runserver
```
